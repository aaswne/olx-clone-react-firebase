# OLX Clone

A full-stack **OLX-style marketplace web application** built using **React.js and Firebase**.  
Users can create accounts, log in, and post product listings similar to the OLX platform.

---

## Features

- User signup and login using Firebase Authentication
- Post product listings with product details
- View products posted by other users
- Dynamic product rendering using Firebase Firestore
- Responsive UI for desktop and mobile
- Reusable React components

---

## Tech Stack

- React.js
- Firebase Authentication
- Firebase Firestore
- JavaScript (ES6+)
- HTML5
- CSS3

---

## Project Inspiration

The UI design of this project is inspired by the **OLX marketplace platform**.  
This project was developed for **learning React and Firebase integration**.

---

# How to Use

1. Sign up or log in using your email.
2. Browse products listed by other users on the homepage.
3. Click the **Sell** button to post a new product.
4. Enter product details such as title, price, and description.
5. Submit the form to add the product to the marketplace.
6. Newly added products will appear instantly on the homepage.
